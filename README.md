# Gem-Game

A simple browser-based gem collection game built with vanilla JavaScript and CSS.

**Live Demo:** [gem-game.vercel.app](https://gem-game.vercel.app)

## About

Gem-Game is a lightweight web game developed as a school project to practice DOM manipulation, event handling, and game loop logic using pure JavaScript — no frameworks or libraries.

## Tech Stack

- **HTML** — game structure
- **CSS** — styling and animations  
- **JavaScript** — game logic, DOM manipulation, sound effects

## Features

- Gem collection mechanics
- Sound effects
- Deployed on Vercel
